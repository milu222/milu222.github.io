    19.盒子的隐藏/超出隐藏

    麋鹿 3/1/2023 CSS

    （1）方式一：

    overflow：hidden;   // 仅仅是隐藏盒子超出的部分
    1

    （2）方式二：

    display: none;	  // 隐藏盒子，而且不占位置 (用的最多)一
    1

    般用于广告的点击后消除效果 (用的最多)

    （3）方式三：

    visibility: hidden;   
visibility: visible;   //让盒子重新显示
    1
    2

    隐藏盒子，但是需要占据实际的位置。

    （4）方式四：

    opacity: 0;       //设置盒子的透明度（不建议，因为内容也会半透明），占位置
    1

    //把盒子移得远远的，占位置。
